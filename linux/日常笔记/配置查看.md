### 查看 ubuntu 服务器的配置

1. cpu 型号

   ```
   lscpu
   cat /proc/cpuinfo | grep name | cut -f2 -d: | uniq -c 
   ```

2. 查看显卡配置

   ```
   # 如果装了英伟达驱动
   nvidia-smi -a 
   # 如果没有安装的话
   lspci | grep vga  # 所有显卡
   lspci -v -s 00:02.0  # 输入显卡序号，获取某块显卡的详细信息
   ```

3. 硬盘大小

   ```
   fdisk -l | grep Disk 
   lsblk   # 查看几块硬盘
   cat /sys/block/磁盘名字/queue/rotational   # 是否固定。返回 0 为 SSD 盘；返回 1 为 SATA 盘 
   du -sh /home/*  # 各个用户的硬盘占有情况
   ```
