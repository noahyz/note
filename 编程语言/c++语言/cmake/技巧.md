#### 2. 获取指定目录下的文件

```
// 会把 ./src 目录下所有符合 *.cpp 结尾的文件存入 USER_LIBS_PATH 变量中
file(GLOB USER_LIBS_PATH ./src/*.cpp)

// 递归寻找指定目录下的所有 *.cpp 文件
file(GLOB_RECURSE USER_LIBS_PATH ./src/*.cpp)
```
