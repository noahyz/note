#### 一、vector 中的 erase 使用

erase 可以删除指定元素，或者删除一个范围的元素。然后迭代器指向下一个元素。

```
for (it = vec.begin(); it != vec.end();) {
	if (xxx) it = vec.erase(it);
	else ++it;
}
```
