## Git 使用方法

#### 1. 比较两个版本之间的区别

```shell
1. 查看当前没有 add 的内容修改
git diff
2. 查看已经 add 但没有 commit 的改动
git diff --cached
3. 查看任意两个版本之间的改动
git diff 版本1 版本2
4. 查看两个版本的 src 文件夹的区别
git diff 版本1 版本2 src 
```

#### 2. 提交

```shell
1. 查看commit内容
git show commit_id

2. 查看最近n次提交的修改，指定n为1则可以查看最近一次修改的内容
git log -p -n
```
